# ExamOnline - 在线考试系统

一个基于Java Web的在线考试系统，支持学生、教师、管理员三种角色。

## 功能特性

### 学生端
- 在线答题（判断题、单选题、多选题、简答题、编程题）
- 实时倒计时
- 自动交卷
- 成绩查询

### 教师端
- 试卷批阅
- 成绩统计
- 学生管理

### 管理员端
- 用户管理（学生、教师）
- 试题管理
- 考试设置
- 系统配置

## 技术栈

- **后端**: Java Servlet + JSP
- **前端**: HTML5 + CSS3 + JavaScript
- **数据库**: Oracle
- **服务器**: Tomcat

## 界面预览

系统采用现代化的UI设计，包含：
- 渐变色彩主题
- 响应式布局
- 卡片式界面
- 流畅的动画效果

## 安装部署

1. 导入项目到Eclipse/IDEA
2. 配置Oracle数据库连接
3. 部署到Tomcat服务器
4. 访问 `http://localhost:8080/ExamOnline`

## 默认账号

- 管理员: admin/admin
- 学生: 学号/密码
- 教师: 教师号/密码
